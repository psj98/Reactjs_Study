## List & Key

### List & Key 란?

- List
  - 목록 => 배열 사용
  - JS의 변수나 객체들을 하나로 묶어 놓은 것

- Key
  - 각 객체나 아이템을 구분할 수 있는 고유한 값
  - 아이템들을 구분하기 위한 고유한 문자열
  - 같은 List에 있는 Elements 사이에서만 고유한 값이면 됨
  - 고유한 값이 없다면, index 사용할 것 => 성능 저하

<br/>

### 여러 개의 Component 렌더링하기

**map() 함수**

- 반복문으로 사용
